# KMI Local Database Files

This folder contains local JSON database files used by the app.

## Files

### branches.json
Public branch data:
- Branch names
- Countries and cities
- Addresses
- Training days and hours
- Coach IDs

### network_coaches.json
Public coach data:
- Coach / instructor names
- Rank
- Experience
- Training
- Certifications
- Branch IDs
- Description

### membership_paid_users.example.json
Example only.
Do not store real paid member data inside the app assets.

Real paid membership data should be stored in Firestore/server.
